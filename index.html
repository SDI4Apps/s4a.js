Introducing s4a.js

s4a.js is a client-side JavaScript library for rapid application development based on OpenLayers 3 and D3.js. It is developed by the SDI4Apps consortium, co-funded by the European Commission, and is currently in the stage of implementation. The first public release is planned for April 2016.
